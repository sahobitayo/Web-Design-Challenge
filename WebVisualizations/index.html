Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we randomly chose 500 cities to include in our analysis and pulled weather data from the OpenWeather API for those randomized cities.

After assembling the dataset, we used the python library, Matplotlib, toplot various aspects of the weather vs. latitude. Factors we looked at include: Temperature, Humidity, Cloudiness, Wind Speed, and humidity. This site provides the data and visualizations created as part of the analysis, as well as a explanations and descriptions of any trends and correlations witnessed.
